Une présentation rapide du mini-projet


Le mini-projet consiste à écrire des programmes (sous forme de script bash) permettant, à partir d'un fichier de texte contenant des URLs, de récupérer le numéro de ligne du résultat, le code de réponse http des sites, leur URl et leur encodage, et de mettre toutes ses données dans un tableau d'un fichier HTML.
